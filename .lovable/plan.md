
# DevHub - מנהל הפרויקטים שלך 🚀

## עיצוב
- **רקע לבן** עם **מסגרות זהב** (#D4AF37)
- **טקסט ואייקונים בכחול נייבי** (#1B2A4A)
- עיצוב נקי ומקצועי עם פונטים ברורים

## מבנה האפליקציה

### 1. סיידבר ניווט (זהב + נייבי)
- דשבורד, פרויקטים, חשבונות, גיבויים, הגדרות
- אייקוני Lucide, מתכווץ למצב מיני

### 2. דשבורד מרכזי
- **כרטיסי סיכום**: סה״כ פרויקטים, פעילים, דורשים תשומת לב, גיבויים אחרונים
- **רשימת פרויקטים אחרונים** עם סטטוס (פעיל/מושהה/הושלם)
- **טיימליין שינויים אחרונים** מכל הפרויקטים
- **גרף פעילות** שבועי/חודשי

### 3. דף פרויקטים
- **תצוגת Grid/List** עם חיפוש וסינון (לפי פלטפורמה, סטטוס, קטגוריה)
- כל כרטיס פרויקט מציג: שם, תיאור, פלטפורמה (GitHub/מקומי), שפת תכנות, תאריך עדכון, סטטוס
- **תגיות וקטגוריות** מותאמות אישית
- כפתור הוספת פרויקט חדש (ידני או חיבור ל-GitHub repo)

### 4. דף פרויקט בודד
- **פרטי פרויקט**: שם, תיאור, לינקים, טכנולוגיות
- **לוג שינויים**: היסטוריית עדכונים עם תאריכים ותיאורים
- **חשבונות מקושרים**: API keys, שם משתמש, סיסמאות (מוצפנים ומוסתרים)
- **הערות ומשימות** של הפרויקט
- **כפתור גיבוי** ידני + היסטוריית גיבויים

### 5. דף ניהול חשבונות
- רשימת כל החשבונות (GitHub, hosting, domains, etc.)
- שמירה מאובטחת של credentials (הצגה מוסתרת עם כפתור "הצג")
- קישור חשבונות לפרויקטים

### 6. דף גיבויים
- סטטוס גיבויים אחרונים לכל פרויקט
- אפשרות גיבוי ידני ושחזור
- לוג גיבויים עם תאריכים וגדלים

## טכני
- **Supabase** (Lovable Cloud) לאחסון נתונים, אימות משתמשים, ושמירת credentials מוצפנים
- **RLS** לאבטחת מידע לפי משתמש
- טבלאות: projects, accounts, credentials, changelogs, backups, tags
- ממשק RTL מלא (עברית)
